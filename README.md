# cic-factory

Hierarchikus agent factory CIC repókhoz — git-követett job-ok, domain-specializált kontextus.

---

## Gyors megértés AI-val

A rendszer komplexitása miatt ajánlott a `workdir/` könyvtárat egy AI-val (pl. Claude Code) végigolvastatni. Induló kontextus sorrendben:

1. `CLAUDE.md` — működési modell, szerepek, job lifecycle
2. `docs/ecosystem-map.md` — teljes ökoszisztéma térkép
3. `jobs/` — konkrét job példák a lifecycle illusztrálásához

Részletes onboarding: [`docs/onboarding.md`](docs/onboarding.md)
